算法题:成绩排序

4月 02, 2019 发布在 数据结构与算法
题目描述

查找和排序

题目：输入任意（用户，成绩）序列，可以获得成绩从高到低或从低到高的排列,相同成绩都按先录入排列在前的规则处理。
示例：

  • jack 70
  • peter 96
  • Tom 70
  • smith 67
    从高到低 成绩
  • peter 96
  • jack 70
  • Tom 70
  • smith 67
    从低到高
  • smith 67
  • jack 70
  • Tom 70
  • peter 96
    输入描述：

    输入多行，先输入要排序的人的个数，然后输入排序方法0（降序）或者1（升序）再分别输入他们的名字和成绩，以一个空格隔开

    输出描述

    按照指定方式输出名字和成绩，名字和成绩之间以一个空格隔开

    示例1
    输入
    3
0
fang 90
yang 50
ning 70
输出
fang 90
ning 70
yang 50

—
解法—python

python直接使用了自带的sorted函数，并且利用dic很好的解决了类似c/c++中需要构造结构体的尴尬。sorted函数的用法如下：
语法

sorted(iterable, /, *, key=None, reverse=False)

Return a new list containing all items from the iterable in ascending order.

A custom key function can be supplied to customize the sort order, and the
reverse flag can be set to request the result in descending order.

  • iterable –可用来迭代的对象
  • key –用来进行排序的元素，可以从可迭代对象中取得
  • reverse –排序的规则，True为降序 False为生序（默认）
    返回值为排序后的list

解法-c
因为使用了快速排序-非稳定排序，导致排序之后次序会发生改变，所以应该采用别种排序，等待修改。
